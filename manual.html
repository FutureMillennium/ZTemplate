ZTemplate

version 2.0.0

Example usage

  1. Make sure the script can write to the temporary directory (defaults to 'tmp')
  2. Place your template in the 'templates' directory (by default), e.g. templates/example.php: 
    <html>
	<head>
		<title>{$t->title}</title>
	</head>
	<body>
		{$t->content}
	</body>
</html>
  3. Include ZTemplate and call the template: 
    <?php
require 'libs/ztemplate.php';

$t['title'] = 'Example title';
$t['content'] = 'Hello world!';

template('example');

Result

<html>
	<head>
		<title>Example title</title>
	</head>
	<body>
		Hello world!
	</body>
</html>

Template syntax

Comments

Example

{*comment*}
Turns into

(nothing)

Print variable

Example

{$foo}
Turns into
<?php echo $foo ?>

If, else if, else

Example

{if foo}
	Lorem
{elseif bar}
	Ipsum
{else}
	Dolor
{/if}
Turns into
<?php if (foo) { ?>
	Lorem
<?php } elseif (bar) { ?>
	Ipsum
<?php } else { ?>
	Dolor
<?php } ?>

For each

Example

{foreach foo}
	Lorem
{/foreach}
Turns into
<?php foreach (foo) { ?>
	Lorem
<?php } ?>

Template variable shorthand

Example

$t->foo
{$t->foo}
{if $t->foo}
Turns into
$t['foo']
<?php echo $t['foo'] ?>
<?php if ($t['foo']) { ?>

Settings

Templates directory

Defined by the TEMPLATES_DIR constant. Should be defined without the last / (slash), unless it's the root directory '/'.

Defaults to 'templates'.

Example

<?php
define('TEMPLATES_DIR', 'templates');

Current template directory

Defined by the $currentTemplate variable. Should be defined without the last / (slash).

Defaults to ''.

Example

<?php
$currentTemplate = 'foo';

Temporary directory

Defined by the TMP_DIR constant. Should be defined without the last / (slash), unless it's the root directory '/'.

Defaults to 'tmp'.

Example

<?php
define('TMP_DIR', 'tmp');

Check for new template updates

Defined by the CHECK_TEMPLATE_UPDATES constant.

Defaults to true.

Example

<?php
define('CHECK_TEMPLATE_UPDATES', false);
